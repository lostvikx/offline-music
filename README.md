# Offline Music Downloader

A simple python script to download music from a .csv file.

## Prerequisite

```bash
mkdir ~/Music/docs
```

## Run Script

```bash
uv run main.py <path_to_music.csv>  # Example: ~/Music/docs/music.csv
```

## Valid CSV File

| Artist      | Song                  |
|-------------|-----------------------|
| Survivor    | Eye of the Tiger      |
| Tom Petty   | Love Is A Long Road   |
